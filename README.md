# advanced-agent

当我们使用一个框架或者工具时，我们可能会想比较一下相关的其他工具，这个agent会从  

- pricing_model
- is_open_source
- tech_stack
- description
- api_available
- language_support
- integration_capabilities

几个方面进行比较，最后进行总结，推荐一个最好的工具。
